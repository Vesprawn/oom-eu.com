  • Raid Loot

    We use Guild Master Loot in current raid content.

    Mostly it’s Main Spec (MS) > Off Spec (OS) with a few things taken into account to share loot most fairly:

    • Already received loot
    • Needing the baseline token loot>warforged token loot>titanforged token loot
    • Gaining 2 set or 4 set
  • Mythic Raiding

    We do not have plans for mythic raiding, maintaining a 20 man team is more exclusive than we want OoM to be. However this doesn’t mean that you are banned from organising it yourself.

  • Alts

    Alts are welcome to join Out of Mana, just ask for an invite on them.

    We don't necessarily want people to be devoted to a class / spec, but we do want them to be dedicated to a role, especially when it comes to those who want to try out healing. And if you want to learn a new role perhaps using your own initiative for alt raids or something like that would be appreciated - officers shouldn't be responsible for setting up events to help people learn something.

  • Discord

    Feel free to join in! We use it as an extension of guild chat, with some more specialist channels. Don’t worry if you think it might be the wrong channel to post something, the channel names are more of a guide than a rule.

  • Taking a Break?

    Please inform an officer if you're going to be away for longer than a month, to avoid being kicked.
